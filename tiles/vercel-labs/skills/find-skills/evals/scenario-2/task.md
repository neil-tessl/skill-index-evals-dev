# Finding an Agent Skill for Regulatory Compliance Document Parsing

## Problem/Feature Description

A legal-tech startup is building a compliance review tool that needs to parse and interpret highly specialized regulatory documents — specifically, financial audit trail reports in the XBRL taxonomy format used by securities regulators. The team has heard about the open agent skills ecosystem and wonders if there's already a skill that covers this domain, which would save them significant setup time.

A project manager has asked you to check whether any existing agent skill covers XBRL financial document parsing or securities regulatory compliance analysis, and to produce a written response they can share with the team. The response should be honest about what was found (or not found), and should suggest a practical next step.

## Output Specification

Produce a file called `skills_search_response.md` that contains:
- A description of what you searched for and how
- An honest assessment of what you found
- A clear recommendation for how the team should proceed

Clean up any large downloaded files before finishing.
